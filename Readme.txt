Dependências

Ruby 2.4.0p0
Rails 5.0.2
Bundler 1.14.3

Foi utilizado a linguaguem ruby, framework rails e bundler. 

Utilizando API:

Deve subir o servidor da API com o comando 'rails s' após ser copiado no local de preferencia. (servidor utilizado: Puma)
Link para acesso do servidor : http://localhost:3000


A aplicação  encontra-se na pasta locaweb_api.

rails server

O servidor backend pode ser acessado pelas urls:

http://localhost:3000/most_relevants - Lista os tweets por relevância.


http://localhost:3000/most_mentions - Lista os usuários que mais mencionam a Locaweb.
